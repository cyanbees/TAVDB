六、当前正式版播放策略
正式版第一版建议：
只返回 Auto master m3u8
不要拆 1080P / 720P / 480P
原因：
Auto：有声音，播放成功，普通拖动正常，清晰度观感至少 720p 以上。
1080P / 720P / 480P：能播放但无声音，不适合作为正式源。
playerType 策略：
目前 Auto 不固定 playerType 也有声音。
所以正式版 v0.1 可以暂时不固定 playerType。
如果后续不同源或不同机型出现无声，再尝试加 playerType: "ijk"。

八、核心函数草案
1. 提取 serverMap
function extractServerMap(html) {
  const map = {};
  const re = /data-link=["']([^"']+)["'][^>]*>\s*([^<]+?)\s*<\/a>/gi;

  let m;
  while ((m = re.exec(html)) !== null) {
    const dataLink = String(m[1] || "").trim();
    const name = String(m[2] || "").trim();

    if (name && dataLink) {
      map[name] = dataLink;
    }
  }

  return map;
}
2. 反转 data-link
function reverseString(s) {
  return String(s || "").split("").reverse().join("");
}
3. 从 supremejav API 返回 HTML 提取 m3u8
function extractM3U8(text) {
  const s = String(text || "")
    .replace(/\\\//g, "/")
    .replace(/&amp;/g, "&");

  const m1 = s.match(/urlPlay.*?(https?:\/\/[^"'\\\s<>]+?\.m3u8[^"'\\\s<>]*)/i);
  if (m1 && m1[1]) return m1[1];

  const m2 = s.match(/https?:\/\/[^"'\\\s<>]+?\.m3u8[^"'\\\s<>]*/i);
  if (m2 && m2[0]) return m2[0];

  return "";
}
4. 从 TV data-link 获取 master m3u8
async function getM3U8ByDataLink(dataLink) {
  const c = reverseString(dataLink);
  const apiUrl = `https://lk1.supremejav.com/supjav.php?c=${encodeURIComponent(c)}`;

  const resp = await Widget.http.get(apiUrl, {
    headers: {
      "User-Agent": UA,
      "Referer": "https://supjav.com/",
      "Origin": "https://supjav.com",
      "Accept": "text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8",
      "Accept-Language": "zh-CN,zh;q=0.9,en;q=0.8"
    },
    timeout: 15000
  });

  if (!resp || resp.statusCode !== 200) {
    throw new Error(`supremejav API HTTP ${resp?.statusCode || "unknown"}`);
  }

  return extractM3U8(resp.data || "");
}
5. headers
function headersFor(url) {
  let referer = "https://supjav.com/";

  try {
    referer = new URL(url).origin + "/";
  } catch (e) {}

  return {
    "Referer": referer,
    "User-Agent": UA,
    "Accept": "*/*"
  };
}
6. stream item
function buildStreamItem(masterM3U8, detailUrl) {
  return {
    name: "Supjav Auto",
    description:
      `来源：Supjav TV\n` +
      `模式：Auto master playlist\n` +
      `详情页：${detailUrl}`,
    url: masterM3U8,
    customHeaders: headersFor(masterM3U8)
  };
}

## 十、2026-06-04 音频问题调查结论

**现象：** 1080P/720P/480P 子播放列表直接播放无声，只有 Auto (master playlist) 有声音。

**调查过程：**
1. 重新请求 master m3u8 `https://cdn2.turboviplay.com/data1/6a0c1c9b5cfd2/6a0c1c9b5cfd2.m3u8`，内容只有 3 个 `#EXT-X-STREAM-INF` 行，无 `#EXT-X-MEDIA:TYPE=AUDIO` 音轨组声明。
2. 观察各变体码率：
   - "480P" → BANDWIDTH=52800 (52.8 kbps) — ⚠️ 太低，不可能是视频
   - "720P" → BANDWIDTH=105600 (105.6 kbps) — ⚠️ 也太低
   - "1080P" → BANDWIDTH=1205600 (1.2 Mbps) — ✅ 正常视频码率
3. **推论：** 480P/720P 条目实际上是**音频轨**的伪装（带有 `RESOLUTION` 属性但码率对应 AAC 音频），1080P 是纯视频轨。
4. 播放器加载 master 时，会自动将最高码率的视频轨（1080P）与其中一个音频轨合并播放 → 有声音。
5. 直接加载 1080P 子播放列表时，播放器没有音频轨信息 → 视频轨无声。

**结论：模块必须返回 master m3u8 URL，不能拆分子播放列表。**
